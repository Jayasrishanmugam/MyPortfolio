# Professional Portfolio Website

A modern, responsive portfolio website built with React and Bootstrap, designed to showcase my work as a Data Analyst. This project demonstrates my skills in data analysis, visualization, and web development.


## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface built with React and Bootstrap 5
- **Smooth Navigation**: Single-page application with smooth scrolling between sections
- **Interactive Components**: Hover effects, transitions, and micro-interactions for enhanced user experience
- **Contact Form**: Functional contact section for potential clients and employers
- **Project Showcase**: Dynamic project cards with links to GitHub repositories
- **Skills Display**: Comprehensive skills section highlighting technical expertise

## Tech Stack

### Frontend
- **React 19.1.1** - Modern JavaScript library for building user interfaces
- **Bootstrap 5.3.7** - Popular CSS framework for responsive design
- **React Scripts** - Build tooling and development environment
- **Axios** - HTTP client for API interactions

### Development Tools
- **Create React App** - Bootstrapped with Facebook's official React starter
- **ESLint** - Code quality and linting
- **Jest & React Testing Library** - Testing framework and utilities

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jayasrishanmugam/MyPortfolio.git
   cd MyPortfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
